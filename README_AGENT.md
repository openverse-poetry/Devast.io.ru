# AI Terminal Agent

Автономный AI агент для работы в терминале с возможностью выполнения команд и редактирования файлов.

## Возможности

- **ShellTool** - выполнение bash команд в терминале
- **ReadFile** - чтение файлов проекта
- **WriteFile** - запись и исправление файлов (код, конфиги)
- **ListDirectory** - просмотр содержимого директорий

## Требования

1. **Python 3.8+**
2. **Ollama** - локальная LLM платформа

## Установка

### 1. Установите зависимости Python

```bash
pip install langchain langchain-community langchain-ollama
```

### 2. Установите Ollama

```bash
# Linux/Mac
curl -fsSL https://ollama.ai/install.sh | sh

# Или скачайте с https://ollama.ai
```

### 3. Запустите Ollama и скачайте модель

```bash
# В одном терминале запустите сервер
ollama serve

# В другом терминале скачайте модель
ollama pull llama3.2
```

Доступные модели:
- `llama3.2` - легкая и быстрая (рекомендуется)
- `mistral` - хорошая для кода
- `codellama` - специализированная для программирования

## Запуск агента

```bash
python ai_agent_terminal.py
```

## Как это работает

1. **Вы вводите задачу** на естественном языке
2. **Агент анализирует** запрос и планирует действия
3. **Агент вызывает инструменты** последовательно:
   - Читает файлы при необходимости
   - Выполняет команды в терминале
   - Исправляет ошибки в коде
4. **Агент докладывает** о результатах каждого шага
5. **Цикл продолжается** пока задача не будет решена

## Конфигурация

В файле `ai_agent_terminal.py` можно изменить:

```python
# Рабочая директория
WORK_DIR = "/workspace"

# Модель Ollama
llm = ChatOllama(
    model="llama3.2",  # Измените на другую модель
    temperature=0,
    base_url="http://localhost:11434"
)

# Лимит шагов выполнения
max_steps = 10
```

## Безопасность

⚠️ **Внимание**: Агент имеет доступ к выполнению команд в вашем терминале!

- Агент ограничен рабочей директорией (`WORK_DIR`)
- Не запускайте агент с правами root без необходимости
- Проверяйте команды перед критическими изменениями

## Команды управления

- `exit` / `quit` / `выход` - завершить работу
- `clear` - очистить историю диалога
- `help` - показать справку
